# C Compiler

A C compiler written in C, built incrementally following [Nora Sandler's Writing a C Compiler](https://norasandler.com/2017/11/29/Write-a-Compiler.html) series.

## Stages
- [ ] Stage 1 — Return an integer
- [ ] Stage 2 — Unary operators
- [ ] Stage 3 — Binary operators
- [ ] Stage 4 — Local variables
- [ ] Stage 5 — Conditionals
- [ ] Stage 6 — Compound statements
- [ ] Stage 7 — Loops
- [ ] Stage 8 — Functions
- [ ] Stage 9 — Global variables
- [ ] Stage 10 — Pointers

## Toolchain
- Written in C
- Targets x86-64 Linux assembly (AT&T syntax)
- Assembled with GCC
